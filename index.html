Automated Measurement System

Welcome to the Automated Measurement System

In this project, we've developed an automated system capable of measuring various parameters and providing real-time updates.

Components

  • Raspberry Pi
  • Ultrasonic Sensor
  • LM35 Temperature Sensor
  • GSM Module
  • LCD Display
  • Load Cell Sensor

How It Works

  1. Height Measurement: The ultrasonic sensor emits sound waves and measures their reflection to calculate height.
  2. Temperature Sensing: The LM35 sensor captures ambient temperature data.
  3. SMS Notification: The GSM module sends real-time updates via SMS.
  4. Data Logging: Measurements are logged into Google Sheets for analysis.
  5. Weight Measurement: The load cell sensor measures weight using ADC.
